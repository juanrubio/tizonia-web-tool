{% trans "About Tizonia OpenMAX IL" %}

{% blocktrans %}Tizonia OpenMAX IL is an experimental implementation for Linux of the OpenMAX IL 1.2 provisional specification from The Khronos Group.{% endblocktrans %}

{% blocktrans %} The project currently includes a number of libraries written in C or C++, including an OpenMAX IL base component library, an implementation of the OpenMAX IL Core, and OS abstraction/utility library, a resource management framework and a number of OpenMAX IL components, like an MP3 decoder, a VP8 decoder, and audio and video renderers. {% endblocktrans %}

{% blocktrans %}Tizonia OpenMAX IL is released under the GNU Lesser General Public License version 3 license.{% endblocktrans %}

{% trans "About Skema, Tizonia's test execution framework" %}

{% blocktrans %}Skema is a test execution framework written in Python that can be used to create arbitrary multimedia graphs and use cases to exercise the Tizonia OpenMAX IL components. {% endblocktrans %}

{% blocktrans %}Skema borrows ideas and some of its code from Linaro's excellent Lava-test framework. {% endblocktrans %}

{% blocktrans %}Skema is released under the GNU General Public License version 3 license.{% endblocktrans %}
